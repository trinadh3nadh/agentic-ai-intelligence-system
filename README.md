# Agentic AI Intelligence System

A multi-agent AI platform capable of:

• Autonomous research
• Multi-agent debate reasoning
• Automated dataset analysis
• Knowledge graph extraction
• Memory-based reasoning

Technologies used:
- LLM orchestration
- Retrieval augmented reasoning
- Vector memory
- Autonomous agents

Built using open-source tools including:
Groq LLM, FAISS, SentenceTransformers, Streamlit.
